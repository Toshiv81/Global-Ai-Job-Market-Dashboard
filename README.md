# 🚀 AI Job Market Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an end-to-end Power BI dashboard analyzing the global AI job market.  
It focuses on salary trends, skills demand, industry insights, and remote work patterns.

The goal of this project is to transform raw job data into meaningful insights that help understand how the AI job market is evolving.

---

## 🎯 Objectives
- Analyze salary trends across experience levels  
- Identify the most in-demand AI skills  
- Explore industry-wise hiring patterns  
- Understand the impact of remote work on salaries  
- Visualize global job distribution  

---

## 📊 Dashboard Features

### 🟦 Page 1: Overview
- Total jobs, average salary, and experience KPIs  
- Job distribution by experience level  
- Salary comparison across experience  
- Employment type and work mode overview  

---

### 🟩 Page 2: Salary Analysis
- Salary growth with experience  
- Top 10 highest paying job roles  
- Salary distribution across ranges  
- Salary comparison by company size  

---

### 🟨 Page 3: Skills & Industry
- Top in-demand AI skills  
- Skill vs salary analysis  
- Industry hiring heatmap  
- Industry-wise salary comparison  

---

### 🟪 Page 4: Location & Remote
- Global hiring map  
- Top hiring countries  
- Remote vs onsite job distribution  
- Work mode impact on salaries  

---

## 💡 Key Insights
- Average AI salaries range between $100K–$110K, with senior roles earning significantly higher  
- Top skills like Python and Machine Learning dominate job requirements  
- High-paying roles exceed $120K+, especially in specialized AI domains  
- Remote and hybrid roles make up a large share of opportunities (50%+)  
- Certain industries offer both high demand and high compensation

## 🛠️ Challenges & Solutions

| Challenge | Solution |
|----------|----------|
| Mixed currencies (USD, EUR) | Converted all salaries into USD for consistency |
| Skills column not structured | Split skills into rows for accurate analysis |
| Overcrowded visuals | Applied Top N filtering and simplified layout |
| Map visualization issues | Cleaned location data and optimized map settings |
| Basic dashboard look | Redesigned using modern layout and better visuals |
